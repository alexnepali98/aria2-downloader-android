1. VERIFY ENVIRONMENT

   - Install Android SDK (API 34)
   - Set ANDROID_HOME environment variable
   - Verify JDK 11 installation

2. NAVIGATE TO PROJECT
   cd /home/node/.openclaw/workspace/Aria2DownloaderAndroid

3. BUILD THE PROJECT
   ./gradlew assembleDebug

4. INSTALL ON DEVICE
   adb install app/build/outputs/apk/debug/app-debug.apk

5. LAUNCH THE APP
   - On device: Tap Aria2 Downloader icon
   - Via ADB: adb shell am start -n com.aria2.downloader/.MainActivity

6. TEST FEATURES
   - Create a new download with a valid URL
   - Test pause/resume
   - Check download history
   - Verify notifications
   - Test settings

7. CUSTOMIZE (Optional)
   - Update app name in res/values/strings.xml
   - Change icon in res/drawable/ic_launcher_foreground.xml
   - Modify theme in ui/theme/Color.kt
   - Adjust settings in SettingsViewModel

8. RELEASE TO PRODUCTION

   - Configure ProGuard rules (already included)
   - Build release APK: ./gradlew assembleRelease
   - Sign APK with keystore
   - Upload to Google Play Store
